## Run Locally

Clone the project

```bash
  git clone https://github.com/jibantajpuriyaa/Todo-website.git
```

Go to the project directory

```bash
  cd Todo-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
